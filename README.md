# NotchFix

> 解决 macOS 刘海屏菜单栏图标被隐藏的问题

## 问题

macOS M4 刘海屏会压缩菜单栏空间，当图标过多时，系统会自动隐藏部分图标，导致无法访问常用工具。

## 解决方案

NotchFix 是一个原生 macOS 菜单栏管理工具，提供：

- 🎯 隐藏不常用的菜单栏图标
- 📦 通过"更多"按钮快速访问隐藏的图标
- ⚡️ 轻量级，原生 Swift 实现
- 🔒 完全本地运行，无需网络权限

## 技术栈

- Swift 5.9+
- SwiftUI
- macOS 13.0+
